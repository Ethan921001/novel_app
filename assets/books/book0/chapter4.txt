第四章：卷哥震怒，學姐傻眼
兩週後，全班第一次小考結束。

題目是：

請寫出一個程式輸入一個字串，判斷是否為回文。

解釋什麼是指標與指標陣列的差異。

簡單、溫和，對剛入門的大一新生來說仍然有如晴天霹靂。但龍傲天寫完不到十分鐘就交卷，還在答案欄空白處用C語言寫了一段自介程式：

c
複製
編輯
#include <stdio.h>
int main() {
    printf("我是你們的爸爸，龍傲天。\\n");
    return 0;
}
隔天成績公布，全班平均 42 分，卷哥 95，龍傲天——100。

全班陷入沉默。教授驚嘆：「終於有學生超越卷哥了……！」

卷哥咬牙：「他一定是作弊。」

但學姐們已經注意到了他，尤其是那位資工系女神——程語晴。她瞇起眼睛看著龍傲天，心想：

「這人，值得我抄筆記。」
